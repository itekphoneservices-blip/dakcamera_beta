# DAKCAMERA Beta Application

**Project:** DAKCAMERA  
**Company:** CIPHER TECH  
**Version:** Beta 1.0  
**Users:** Limited to 20 beta testers  
**Deadline:** 1 week from activation

---

## Overview

DAKCAMERA is a private beta application designed by CIPHER TECH for testing the next-generation mobile camera experience. The app allows selected users to submit application forms securely, generate a PDF version automatically, and send it via WhatsApp to the company.  

This app is built with **HTML, CSS, JavaScript** for frontend and **Python (FastAPI)** for backend functionality. It includes security measures to prevent multiple submissions from the same device or IP, ensures only 20 users can submit applications, and enforces a one-week application window.  

---

## Features

- **Private Beta:** Limited to 20 users.  
- **Device/IP Check:** Prevents multiple submissions.  
- **PDF Auto-Generation:** Form submitted is instantly converted to PDF.  
- **WhatsApp Integration:** Automatically opens WhatsApp with pre-filled message.  
- **Security:** Device fingerprinting and backend validation.  
- **Deadline Enforcement:** Application window is exactly 1 week.  
- **Modern UI:** Stylish and flagship-quality frontend.
